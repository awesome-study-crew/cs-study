# 규모에 따른 네트워크 구분

## LAN (Local Area Network)

- 근거리 통신망
- 가정이나 기업처럼 비교적 가까운 거리를 연결하는 한정된 공간에서의 네트워크
- 특정 공유기에 연결된 네트워크 기기들은 모두 같은 네트워크(LAN)에 속해있다고 인식

## CAN (Campus Area Network)

- 여러 개의 LAN을 하나로 묶은 형태로, 주로 대학교 캠퍼스나 대기업 단지 내에서 사용
- LAN 보다는 넓고 MAN 보다는 좁은 범위로 가진다.
- 특정 조직(학교, 기업 등)이 소유하고 관리하며, 건물 간의 데이터 공유를 목적으로 함.
- 대학교 내의 공학관, 도서관 등이 포함.

## MAN (Metropolitan Area Network)

- 한 도시 전체를 연결하는 수준의 네트워크. 여러 개의 LAN이나 CAN이 모여 형성된다.
- 약 5km ~ 50km 정도의 범위를 커버하며, 도시 내 관공서나 지점들을 연결할 때 사용된다.
- 서울시 전체를 아우르는 공공 와이파이망 등이 포함.

## WAN (Wide Area Network)

- 국가나 대륙을 넘나드는 가장 넓은 범위의 네트워크.
- WAN이 인터넷을 가능하게 만드는 네트워크.
- 일반적으로 ISP(Internet Service Provider)라는 인터넷 서비스 업체가 구축하고 관리함.
